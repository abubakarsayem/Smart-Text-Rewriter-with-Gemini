## Smart Text Rewriter with Gemini
A simple web application that rewrites text in different styles using the Gemini API. Users can enter their API key, paste their text, choose a tone (e.g., formal or casual), and select a dialect (e.g., American English). The app then generates a polished rewritten version instantly.

## Features
- Rewrite text in multiple tones.
- Support for different English dialects.
- Uses Gemini API for high-quality results.
- Clean and user-friendly interface.

## How to Use
- Enter your Gemini API key.
- Paste or type your text.
- Select your preferred tone.
- Choose the dialect.

## Use Cases
- Improving writing clarity.
- Converting text to formal or casual tone.
- Adapting content for different audiences.
- Quick editing and polishing.

## Requirements
- A valid Gemini API key.
